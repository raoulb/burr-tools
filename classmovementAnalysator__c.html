Detailed Description

this class is can do analysis of movements within a puzzle.

It takes a position of pieces and then tries to find all other positions that can be reached directly from that starting point.

It is implemented using Bill Cutlers algorithm, so please read there in case you are interested how it works. The comments are written with the thought that you know his algorithm
