Detailed Description

Hashtable like nodeHash with the additional feature of scanning through all elements.

this is a hashtable that stores nodes but is also allows traversal of all nodes hat are within the table at a given point in time, only one such traversal can be active at one time and the nodes are scanned in the reverse order they were added
